# Movie Booking System - Frontend

React.js frontend for the movie booking system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The application will open at http://localhost:3000

## Configuration

Make sure the backend server is running on http://localhost:5000

## Components

- `MovieList` - Display available movies
- `SeatSelection` - Select seats for a movie
- `Checkout` - Complete the booking with payment details

## Features

- Browse available movies
- Select movie seats in a visual seat map
- Provide customer details and payment information
- Booking confirmation
